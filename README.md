# Mvvm test project

Simple project with MVVM+Coordinators architecture.

## Installation

1. Clone project:

    ```bash
    $ git clone git@github.com:surfstudio/MvvmUIKitExample.git
    ```
2. Open `MvvmExampleProject.xcodeproj` in `Xcode`

## Requirements

- Xcode version: `13.2.1` or higher
- Swift toolchain `5.5`
